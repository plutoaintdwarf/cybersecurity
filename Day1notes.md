## Cybersecurity:
Cybersecurity is the practice of protecting computers, networks, and data from:
- unauthorized access (hackers)
- data theft
- malware
- attacks
- system failures

Cybersecurity = Keeping people safe online

## CIA Triad
- Confidentiality: only authorized users can access data
- Integrity: keep data accurate and unchanged
- Availability: systems must be accessible when needed

## Red / Blue / Purple Teams
- Red Team: Offensive, ethical hacking, Penetration Tester
- Blue Team: Defensive, SOC Analyst, Cybersecurity Analyst, Incident Response, Digital Forensics
- Purple Team: Mix of Red + Blue, collaboration

## Key Cybersecurity Roles
- SOC Analyst: monitors alerts and suspicious activity
- Penetration Tester: finds weaknesses by simulating attacks
- Cybersecurity Analyst: handles incidents, improves security
- Incident Response: responds to attacks
- Digital Forensics: finds evidence after attacks

## Basic Terms
- Threat, Vulnerability, Exploit, Malware, Phishing, Firewall , Hashing, Ransomware, Social Engineering.

## OWASP ( Open Web Application Security Project )
The Open Web Application Security Project, or OWASP, is an international non-profit organization dedicated to web application security. 
# OWASP Top 10 = best known project of OWASP
  Regularly updated report outlining security concerns for web application security, focusing on the 10 most critical risk.
- Security risk reported in OWASP Top 10 2021 report:
   - Broken Access control
   - Cryptographic failures
   - Injection
